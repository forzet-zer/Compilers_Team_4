# Requerements:

Python 3 [ https://www.python.org/downloads ]
NLTK (Natural Language Tool Kit) [ https://www.nltk.org/install.html ] [ pip install nltk ]
RPLY [ https://rply.readthedocs.io/en/latest/ ] [pip install rply]


To execute the compiler, execute the file: main.py

The compiler will ask for the name of the program to analyze, which has to be located in the programas folder, if no name is typed,
the compiler will take as default the program holamundo.jp

Afterwards the compiler will ask the user for the number of the stage up to which to execute the compiler, or otherwise can type "exit" to terminate the program.
